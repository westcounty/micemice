# Micemice 墙贴 SOP（半页极简版）v1.0

## 开工 30 秒

1. 看工作台：先清关键/逾期任务
2. 看通知：先处理合规预警
3. 扫笼码：进入笼卡执行

## 现场执行 5 步

1. 扫码或手输笼号进入笼卡
2. 执行转笼/并笼/拆笼或批量状态
3. 个体详情补事件（体重/健康/处理）
4. 必要时上传附件（选文件/拍照）
5. 完成后回任务页标记完成

## 角色分工

1. 实验员：执行+留痕
2. PI：看报表+风险复盘
3. 管理员：主数据/权限/审计/导出

## 红线

1. 协议过期或无协议：关键操作禁止
2. 非法状态跳转：系统阻断
3. 无权限：找管理员开 RBAC 权限点

## 异常兜底

1. 扫码失败：手动输入笼号
2. 自动退出：30 分钟超时，重新登录
3. 导出找不到：按提示路径查 CSV/PDF/ZIP

## 收工 3 项

1. 关键任务已清零或有解释
2. 通知已读
3. 当日导出已备份
