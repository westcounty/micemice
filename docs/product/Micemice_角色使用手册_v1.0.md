# Micemice 角色使用手册 v1.0

- 版本：v1.0
- 日期：2026-02-07
- 适用范围：`Micemice App（离线优先，单机本地）`
- 目标读者：博士生/实验员、PI、管理员

## 1. 使用前须知

1. 当前为 App-only 版本，业务数据保存在本机（无后端依赖）。
2. 登录规则：
   1. 账号：非空
   2. 密码：至少 4 位
   3. 组织编码：非空（默认可填 `LAB-A1`）
3. 会话规则：30 分钟无操作会自动登出。
4. 角色可在管理中心切换，且支持细粒度权限点开关（RBAC）。

## 2. 角色总览

## 2.1 博士生/实验员（Researcher）

主要目标：完成现场高频执行动作，保证记录完整。

常用页面：
1. 工作台：看今日待办、未读通知、快捷入口
2. 扫码：扫码进笼卡
3. 笼位：转笼、并笼、拆笼、批量状态
4. 个体：状态更新、事件记录、附件上传
5. 繁育：配种、查栓、产仔、断奶向导
6. 分型：采样、建批、导入、冲突确认
7. 实验：建实验、记录事件、归档
8. 任务：完成任务、批量处理

## 2.2 PI（Principal Investigator）

主要目标：评估课题执行质量、风险与资源效率。

常用页面：
1. 报表：占用率、任务完成率、繁育成功率、存活率、趋势图
2. 工作台：关键预警和执行概览
3. Cohort：条件筛选、锁定、盲法编码导出
4. 通知中心：协议到期、逾期任务、容量风险

## 2.3 管理员（Admin）

主要目标：主数据治理、合规审计、权限配置、导入导出。

常用页面（管理中心）：
1. 细粒度权限矩阵（按角色/权限点开关）
2. 主数据维护（品系/基因型）
3. 协议启停与到期管理
4. 培训资质维护
5. 通知策略
6. 同步队列与冲突中心
7. 导入导出（CSV/PDF/ZIP）
8. 笼码二维码生成与批量导出
9. 审计查询（含 Before/After 结构化字段）

## 3. 默认权限点说明

说明：管理员可在“细粒度权限矩阵”调整，以下是默认最小角色门槛。

1. Researcher 及以上：
   1. 完成任务
   2. 转笼/并笼/拆笼
   3. 更新个体状态
   4. 记录个体事件
   5. 添加个体附件
   6. 繁育写操作
   7. 分型写操作
   8. Cohort 写操作
   9. 实验写操作
2. Admin 默认专有：
   1. 任务模板与升级规则
   2. 主数据维护
   3. 创建笼位与个体
   4. 协议管理
   5. 培训资质管理
   6. 通知策略管理
   7. 同步与重试
   8. 导入导出
   9. 权限矩阵管理

## 4. 角色操作手册

## 4.1 博士生/实验员日常 SOP

1. 登录并选择 `博士生/实验员`
2. 打开工作台，优先处理：
   1. 逾期任务
   2. 关键优先级任务
   3. 协议临期提醒
3. 进入扫码页，扫笼码进入笼卡
4. 在笼卡执行转笼/并笼/拆笼或批量状态更新
5. 在个体详情补充：
   1. 体重/健康/处理事件
   2. 附件（文件选择或拍照直传）
6. 如有繁育任务，按配种-查栓-产仔-断奶向导完成
7. 如有分型任务，按采样-建批-导入-冲突确认闭环
8. 在任务页批量完成并确认通知已读

## 4.2 PI 周期复盘 SOP

1. 登录并选择 `PI`
2. 打开报表页，设置时间窗（7/30/90 天）和课题过滤
3. 重点看：
   1. 任务完成率趋势
   2. 繁育成功率与存活率
   3. 品系分布与任务状态分布
4. 对异常项回到任务/繁育/实验模块定位具体对象
5. 导出合规资料用于组会或检查准备

## 4.3 管理员治理 SOP

1. 登录并选择 `管理员`
2. 进入管理中心按顺序完成：
   1. 主数据（品系/基因型）
   2. 协议状态与到期策略
   3. 培训资质
   4. 权限矩阵
   5. 通知策略
   6. 生成/导出笼码
3. 每日查看：
   1. 同步队列与冲突中心
   2. 审计记录（按操作者/对象过滤）
4. 每周执行：
   1. 导出 CSV/PDF/ZIP 备份
   2. 检查权限矩阵是否符合实验室制度

## 5. 关键页面说明

1. 工作台：任务与风险聚合入口
2. 扫码：建议优先使用；无法识别可手动输入笼号
3. 笼卡：历史时间线可追溯操作
4. 个体详情：
   1. 状态变更受状态机约束
   2. 关键状态受协议/权限校验
   3. 支持事件与附件双轨记录
5. 管理中心：
   1. 协议启停有二次确认
   2. 审计展示含 Before/After

## 6. 常见问题

1. 进不了管理页面  
原因：当前不是 Admin 或被权限矩阵禁用。  
处理：切角色到 Admin，或由管理员开启权限点。

2. 无法执行某操作，提示“缺少权限”  
原因：细粒度 RBAC 已禁用该权限点。  
处理：管理员进入“细粒度权限矩阵”启用对应能力。

3. 登录后过一段时间被踢出  
原因：30 分钟无操作会话超时。  
处理：重新登录即可。

4. 导出后文件在哪里  
位置：App 外部文件目录（或内部 files 目录），导出成功会显示完整路径。

## 7. 推荐管理制度（建议）

1. Researcher 默认开放执行权限，限制治理权限。
2. PI 开放报表、Cohort、实验复盘相关能力。
3. Admin 每周固定导出一次审计与合规包。
4. 高风险权限（协议管理、权限矩阵）仅保留 1-2 名管理员。
